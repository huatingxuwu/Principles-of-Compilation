============================================================
  测试 1: 经典算术表达式文法
============================================================

────────────────────────────────────────────────────────────
  第一步: 解析文法
────────────────────────────────────────────────────────────
  开始符号: E
  非终结符: ['E', "E'", 'F', 'T', "T'"]
  终结符:   ['(', ')', '*', '+', 'id']
  产生式:
  E → T E'
  E' → + T E'
  E' → ε
  T → F T'
  T' → * F T'
  T' → ε
  F → ( E )
  F → id

────────────────────────────────────────────────────────────
  第二步: 计算 FIRST 集（不动点迭代）
────────────────────────────────────────────────────────────
===== 计算 FIRST 集 =====

  FIRST()) = {)}  （终结符，直接得到）
  FIRST(id) = {id}  （终结符，直接得到）
  FIRST(() = {(}  （终结符，直接得到）
  FIRST(*) = {*}  （终结符，直接得到）
  FIRST(+) = {+}  （终结符，直接得到）

  --- 第 1 轮迭代 ---
    E' → + T E': FIRST(E') 增加 → {+}
    E' → ε: FIRST(E') 增加 → {ε, +}
    F → ( E ): FIRST(F) 增加 → {(}
    F → id: FIRST(F) 增加 → {(, id}
    T' → * F T': FIRST(T') 增加 → {*}
    T' → ε: FIRST(T') 增加 → {ε, *}
  --- 第 2 轮迭代 ---
    T → F T': FIRST(T) 增加 → {(, id}
    E → T E': FIRST(E) 增加 → {(, id}
  --- 第 3 轮迭代 ---

  ◆ 最终 FIRST 集:
    FIRST(E) = {(, id}
    FIRST(E') = {ε, +}
    FIRST(F) = {(, id}
    FIRST(T) = {(, id}
    FIRST(T') = {ε, *}

────────────────────────────────────────────────────────────
  第三步: 计算 FOLLOW 集（不动点迭代）
────────────────────────────────────────────────────────────

===== 计算 FOLLOW 集 =====

  FOLLOW(E) 加入 '$'  （开始符号规则）

  --- 第 1 轮迭代 ---
    F → ( E ): 位置 1 的 E 后跟 ), FOLLOW(E) 增加 → {$, )}
    E → T E': 位置 1 的 E' 后跟 ε（末尾）, FOLLOW(E') 增加 → {$, )}
    T → F T': 位置 0 的 F 后跟 T', FOLLOW(F) 增加 → {*}
    E → T E': 位置 0 的 T 后跟 E', FOLLOW(T) 增加 → {$, ), +}
    T → F T': 位置 1 的 T' 后跟 ε（末尾）, FOLLOW(T') 增加 → {$, ), +}

  --- 第 2 轮迭代 ---
    T → F T': 位置 0 的 F 后跟 T', FOLLOW(F) 增加 → {$, ), *, +}

  --- 第 3 轮迭代 ---

  ◆ 最终 FOLLOW 集:
    FOLLOW(E) = {$, )}
    FOLLOW(E') = {$, )}
    FOLLOW(F) = {$, ), *, +}
    FOLLOW(T) = {$, ), +}
    FOLLOW(T') = {$, ), +}

────────────────────────────────────────────────────────────
  第四步: 构造 LL(1) 预测分析表
────────────────────────────────────────────────────────────

===== 构造 LL(1) 预测分析表 =====

  处理产生式: E → T E'
    FIRST(T E') = {(, id}
    M[E, (] = E → T E'
    M[E, id] = E → T E'
  处理产生式: E' → + T E'
    FIRST(+ T E') = {+}
    M[E', +] = E' → + T E'
  处理产生式: E' → ε
    FIRST(ε) = {ε}
    M[E', $] = E' → ε （ε ∈ FIRST, $ ∈ FOLLOW(E')）
    M[E', )] = E' → ε （ε ∈ FIRST, ) ∈ FOLLOW(E')）
  处理产生式: F → ( E )
    FIRST(( E )) = {(}
    M[F, (] = F → ( E )
  处理产生式: F → id
    FIRST(id) = {id}
    M[F, id] = F → id
  处理产生式: T → F T'
    FIRST(F T') = {(, id}
    M[T, (] = T → F T'
    M[T, id] = T → F T'
  处理产生式: T' → * F T'
    FIRST(* F T') = {*}
    M[T', *] = T' → * F T'
  处理产生式: T' → ε
    FIRST(ε) = {ε}
    M[T', $] = T' → ε （ε ∈ FIRST, $ ∈ FOLLOW(T')）
    M[T', )] = T' → ε （ε ∈ FIRST, ) ∈ FOLLOW(T')）
    M[T', +] = T' → ε （ε ∈ FIRST, + ∈ FOLLOW(T')）

  ◆ 预测分析表:
           $         (         )         *         +        id
  ------------------------------------------------------------
  E                     E→T E'                                  E→T E'
  E'            E'→ε                E'→ε           E'→+ T E'          
  F                    F→( E )                                    F→id
  T                     T→F T'                                  T→F T'
  T'            T'→ε                T'→ε T'→* F T'      T'→ε          

────────────────────────────────────────────────────────────
  第五步: 表驱动预测语法分析
────────────────────────────────────────────────────────────

===== 表驱动预测语法分析 =====
  输入串: id + id * id

  初始栈: [$, E]
  输入: id + id * id $

  步骤 1: 栈顶=E, 当前输入=id, 栈=[$, E]
    应用产生式: E → T E'
    栈更新为: [$, E', T]
  步骤 2: 栈顶=T, 当前输入=id, 栈=[$, E', T]
    应用产生式: T → F T'
    栈更新为: [$, E', T', F]
  步骤 3: 栈顶=F, 当前输入=id, 栈=[$, E', T', F]
    应用产生式: F → id
    栈更新为: [$, E', T', id]
  步骤 4: 栈顶=id, 当前输入=id, 栈=[$, E', T', id]
    匹配终结符 'id'，弹出栈，输入指针前进
  步骤 5: 栈顶=T', 当前输入=+, 栈=[$, E', T']
    应用产生式: T' → ε
    栈更新为: [$, E']
  步骤 6: 栈顶=E', 当前输入=+, 栈=[$, E']
    应用产生式: E' → + T E'
    栈更新为: [$, E', T, +]
  步骤 7: 栈顶=+, 当前输入=+, 栈=[$, E', T, +]
    匹配终结符 '+'，弹出栈，输入指针前进
  步骤 8: 栈顶=T, 当前输入=id, 栈=[$, E', T]
    应用产生式: T → F T'
    栈更新为: [$, E', T', F]
  步骤 9: 栈顶=F, 当前输入=id, 栈=[$, E', T', F]
    应用产生式: F → id
    栈更新为: [$, E', T', id]
  步骤 10: 栈顶=id, 当前输入=id, 栈=[$, E', T', id]
    匹配终结符 'id'，弹出栈，输入指针前进
  步骤 11: 栈顶=T', 当前输入=*, 栈=[$, E', T']
    应用产生式: T' → * F T'
    栈更新为: [$, E', T', F, *]
  步骤 12: 栈顶=*, 当前输入=*, 栈=[$, E', T', F, *]
    匹配终结符 '*'，弹出栈，输入指针前进
  步骤 13: 栈顶=F, 当前输入=id, 栈=[$, E', T', F]
    应用产生式: F → id
    栈更新为: [$, E', T', id]
  步骤 14: 栈顶=id, 当前输入=id, 栈=[$, E', T', id]
    匹配终结符 'id'，弹出栈，输入指针前进
  步骤 15: 栈顶=T', 当前输入=$, 栈=[$, E', T']
    应用产生式: T' → ε
    栈更新为: [$, E']
  步骤 16: 栈顶=E', 当前输入=$, 栈=[$, E']
    应用产生式: E' → ε
    栈更新为: [$]
  步骤 17: 栈顶=$, 当前输入=$, 栈=[$]
    匹配终结符 '$'，弹出栈，输入指针前进

  ✓ 分析成功！输入串是文法的句子。



============================================================
  测试 2: 含 ε 的简单文法
============================================================

────────────────────────────────────────────────────────────
  第一步: 解析文法
────────────────────────────────────────────────────────────
  开始符号: S
  非终结符: ['A', 'B', 'S']
  终结符:   ['a', 'b']
  产生式:
  S → A B
  A → a
  A → ε
  B → b
  B → ε

────────────────────────────────────────────────────────────
  第二步: 计算 FIRST 集（不动点迭代）
────────────────────────────────────────────────────────────
===== 计算 FIRST 集 =====

  FIRST(a) = {a}  （终结符，直接得到）
  FIRST(b) = {b}  （终结符，直接得到）

  --- 第 1 轮迭代 ---
    B → b: FIRST(B) 增加 → {b}
    B → ε: FIRST(B) 增加 → {ε, b}
    A → a: FIRST(A) 增加 → {a}
    A → ε: FIRST(A) 增加 → {ε, a}
  --- 第 2 轮迭代 ---
    S → A B: FIRST(S) 增加 → {ε, a, b}
  --- 第 3 轮迭代 ---

  ◆ 最终 FIRST 集:
    FIRST(A) = {ε, a}
    FIRST(B) = {ε, b}
    FIRST(S) = {ε, a, b}

────────────────────────────────────────────────────────────
  第三步: 计算 FOLLOW 集（不动点迭代）
────────────────────────────────────────────────────────────

===== 计算 FOLLOW 集 =====

  FOLLOW(S) 加入 '$'  （开始符号规则）

  --- 第 1 轮迭代 ---
    S → A B: 位置 0 的 A 后跟 B, FOLLOW(A) 增加 → {$, b}
    S → A B: 位置 1 的 B 后跟 ε（末尾）, FOLLOW(B) 增加 → {$}

  --- 第 2 轮迭代 ---

  ◆ 最终 FOLLOW 集:
    FOLLOW(A) = {$, b}
    FOLLOW(B) = {$}
    FOLLOW(S) = {$}

────────────────────────────────────────────────────────────
  第四步: 构造 LL(1) 预测分析表
────────────────────────────────────────────────────────────

===== 构造 LL(1) 预测分析表 =====

  处理产生式: A → a
    FIRST(a) = {a}
    M[A, a] = A → a
  处理产生式: A → ε
    FIRST(ε) = {ε}
    M[A, $] = A → ε （ε ∈ FIRST, $ ∈ FOLLOW(A)）
    M[A, b] = A → ε （ε ∈ FIRST, b ∈ FOLLOW(A)）
  处理产生式: B → b
    FIRST(b) = {b}
    M[B, b] = B → b
  处理产生式: B → ε
    FIRST(ε) = {ε}
    M[B, $] = B → ε （ε ∈ FIRST, $ ∈ FOLLOW(B)）
  处理产生式: S → A B
    FIRST(A B) = {ε, a, b}
    M[S, a] = S → A B
    M[S, b] = S → A B
    M[S, $] = S → A B （ε ∈ FIRST, $ ∈ FOLLOW(S)）

  ◆ 预测分析表:
           $         a         b
  ------------------------------
  A              A→ε       A→a       A→ε
  B              B→ε                 B→b
  S            S→A B     S→A B     S→A B

────────────────────────────────────────────────────────────
  第五步: 表驱动预测语法分析
────────────────────────────────────────────────────────────

===== 表驱动预测语法分析 =====

    栈更新为: [$, B, a]
  步骤 3: 栈顶=a, 当前输入=a, 栈=[$, B, a]

    应用产生式: B → b

  步骤 5: 栈顶=b, 当前输入=b, 栈=[$, b]
    匹配终结符 'b'，弹出栈，输入指针前进
  步骤 6: 栈顶=$, 当前输入=$, 栈=[$]
    匹配终结符 '$'，弹出栈，输入指针前进

  ✓ 分析成功！输入串是文法的句子。
